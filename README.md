# 图度 - 信息发布平台

## 简介

使用 React-TypeScript + Vite 开发，基于Zustand构建状态管理

## 开发须知

本项目使用 yarn 进行包管理

### 安装依赖

` yarn` 

### 启动项目

` yarn dev` 

### 自动部署

在 main 分支上提交 / Merge 时会触发 Github Action 自动部署，部署状态可见
